# twitter_for_python
<p>CUI環境からツイートできるようにするやつ</p>

<p>使い方</p>

<p>・pip install tweepy</p>
<p>をcmd内で実行する。すでにインストールしてあるなら実行しなくてもよき。</p>
<pp>テキストエディタやらでtwitterfunc/key_config.jsonを開く。</p>
<p>jsonファイルをテキストエディタで開き、自分のアプリのapi keyとapi key secretなどを突っ込む。</p>
<p>突っ込んだら、保存して実行しろください。</p>
<br>
<p>テスト環境</p>
<p>Python 3.8.8</p>
<p>pip 21.0.1</p>
